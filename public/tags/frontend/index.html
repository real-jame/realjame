⚛️ React: When everything looks like a nail...

I’ve been working on textarea.site as a way to dive back into the beautiful and the ugly parts of front-end web development. After a few days of work, I decided to rewrite the entire project in Svelte. Surprising? Maybe not. It’s about the right tool for the job, and here, React is not it. I’m sure people have written countless times about this. But it’s true that being versatile is one of the best skills you can have as a software developer.
